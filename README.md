# FanbridgeThis is a repo for managing machine readable artifacts for the [Fanbridge APIs](http://www.fanbridge.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Fanbridge APIs](http://www.fanbridge.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.fanbridge.com)- [Developer](http://www.fanbridge.com)- [Twitter](https://twitter.com/FanBridge)- [GitHub](https://github.com/fanbridge)The [APIs.json](https://github.com/api-evangelist/fanbridge/blob/master/apis.json) index provides links to all the relevant links for the Fanbridge APIs, including the OpenAPI and Postman if they are present.
